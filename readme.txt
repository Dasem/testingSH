Требования к системе:
1. Установленная Java 8
2. Браузер: (Google Chrome, Internet Explorer 10+ (возможна 9 версия, но с ограниченной функциональностью)
3. Подключение к локальной сети (отсутствие ограничений на это подключение)

Порядок развёртывания и эксплуатации:
1. Запустить файл testing-1.0.4.jar (если сервер не включается в течение длительного времени (1-2 минуты), см. пункт "Возможные ошибки"
2. В меню "Параметры" узнать IP
3. На компьютерах для тестирования открыть браузер, ввести в адресную строку первый из адресов в списке. Если нет подключения, попробовать остальные. Начать тестирование
4. По окончании тестирования результаты будут сохранены в файл result.txt
5. Для выключения сервера закрыть окно

Рекомендации к эксплуатации:
1. Если не установлена Java 8, можно сделать это по ссылке "https://java.com/ru/download/" (для OS linux в зависимости от дистрибутива)
2. При необходимости возможно добавление вопросов в файл questions.xslx (1 столбец: тема, 2: вопрос, 3 и остальные: варианты ответов. У верного/ых ответа/ов необходимо изменить цвет ячейки)
3. Вручную узнать адрес сервера (компьютера, на котором запускается программа), можно с помощью Win+R -> cmd -> ipconfig. IPv4 адрес - это тот адрес, что нужен
4. При возникновении неполадок обращаться по адресу gfhflbuvf46@mail.ru
5. На старых браузерах могут не работать проверки входных данных. Необходимо это регулировать вручную

Рекомендации для дальнейшей поддержки и модификации:
1. Если есть необходимость изменить исходники проекта и перекомпилировать его, для этого есть файл deploy.bat (для его работы необходим Java Development Kit 8 (JDK))
2. Первая компиляция и сборка проекта должна осуществляться с доступом в интернет, так как будут загружаться необходимые библиотеки

Возможные ошибки: 
1. Порт по умолчанию занят. Решение: сменить порт в файле "testingSH\src\main\resources\application.properties" на один из свободных (например, 8080 или 81). После сохранения файла, пересобрать проект (п.1 раздела "Рекомендации для дальнейшей поддержки и модификации")
