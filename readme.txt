1. Запустить файл JavaSetup8u211.exe (если не установлена Java, необходимо подключение к интернету)
2. Запустить файл start.bat
3. Посмотреть строку "Выберите один из адресов для подключения"
4. На компьютерах для тестирования открыть браузер, ввести в адресную строку первый из адресов в списке. Если нет подключения, попробовать остальные. Начать тестирование
5. по окончании тестирования результаты будут сохранены в файл result.txt
